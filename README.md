# Spotify OBS

## Getting Started

- Join Lanyard Discord Server
- Enable Discord Developer Mode and copy your Discord User ID
- Add a Browser Source in OBS with following options:
  - `URL`: `https://obsmusicwidget.vercel.app/:your_discord_id`
  - `Width`: `800`
  - `Height`: `200`
